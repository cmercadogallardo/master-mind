# 🧠 Mastermind - Juego de Lógica

## 📜 Reglas del juego
- El sistema genera un código secreto de **4 colores** (rojo, azul, verde, amarillo, morado, naranja).
- El jugador tiene **10 intentos** para adivinar el código.
- **Pistas:**
  - ✅ **Aciertos**: Color y posición correctos.
  - 🔴 **Coincidencias**: Color correcto en posición errónea.

## 🎮 Cómo jugar
1. Introduce 4 colores en el orden deseado.
2. Recibe retroalimentación inmediata.
3. Ajusta tu estrategia hasta adivinar el código o agotar intentos.
